## Run Locally

**Prerequisites:**  [Android Studio](https://developer.android.com/studio)


1. Open Android Studio
2. Select **Open** and choose the directory containing this project
3. Allow Android Studio to fix any incompatibilities as it imports the project.
4. Create a file named `.env` in the project directory and set `GEMINI_API_KEY` in that file to your Gemini API key (see `.env.example` for an example)
5. Remove this line from the app's `build.gradle.kts` file: `signingConfig = signingConfigs.getByName("debugConfig")`
6. Run the app on an emulator or physical device

## Installation

**Step 1 — Download the APK**

Go to the Actions tab at the top of this page. Select the latest build with a green checkmark. Scroll down to the Artifacts section and click amnvazheh-apk to download a zip file. Extract the zip to get amnvazheh.apk.

**Step 2 — Transfer to your phone**

Send the APK file to your Android phone via USB cable, Telegram, or email.

**Step 3 — Allow installation from unknown sources**

Android 8 and above: Settings → Apps → File manager → Install unknown apps → Allow

Android 7 and below: Settings → Security → Unknown sources → Enable

**Step 4 — Install**

Tap the amnvazheh.apk file and confirm the installation.
